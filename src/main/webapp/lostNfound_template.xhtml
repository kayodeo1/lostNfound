Project Description

A web-based lost and found management system for educational institutions built with JSF, PrimeFaces, and EJB. This application enables students to report lost items, upload found items with photos, and browse through available items with comprehensive search functionality.

Project Motivation

Educational institutions often struggle with managing lost and found items efficiently. Traditional paper-based systems are prone to errors and difficult to manage. Our digital solution provides a centralized platform where students can easily report and search for lost items, while administrators can efficiently manage the entire process.

This system reduces the time spent searching for lost items and increases the likelihood of successful item recovery, creating a more organized and helpful campus environment.

Technology Stack

Frontend: JSF 2.2, PrimeFaces 8.0

Backend: Java EE 7, EJB 3.2

Dependency Injection: CDI 1.2

Server: Apache TomEE Plume 7.1.2

Database: JPA/OpenJPA with H2 for development

Key Features

• View reported lost or found items in a user-friendly grid layout

• Advanced search functionality with keyword filtering

• Easy item reporting system for both lost and found items

• Administrative interface for item approval and rejection

• Image upload and preview capabilities

• Clean, responsive UI design with modern web standards
